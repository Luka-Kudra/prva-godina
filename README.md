# 🗒️ Obsidian Vault: Prva Godina 

> [!TIP]
> Ovaj repozitorijum je optimizovan za **Obsidian**. Da biste videli interne linkove, mape uma i vizuelne dijagrame, preporučujem da ga otvorite direktno u Obsidian aplikaciji.

---

## 🚀 Brza Instalacija

1.  **Instalirajte Obsidian**: [Preuzmi ovde](https://obsidian.md/download)
2.  **Klonirajte repozitorijum**:
    ```bash
    git clone [https://github.com/Luka-Kudra/prva-godina.git](https://github.com/Luka-Kudra/prva-godina.git)
    ```
3.  **Otvorite Vault**: U Obsidianu izaberite `Open folder as vault` i navigirajte do kloniranog foldera.

---

## 🎨Excalidraw Setup

Većina mojih beleški se oslanja na **Excalidraw** fajlove. Bez ovog plugina, videćete samo sirovi JSON kod umesto crteža.

**Da biste pravilno videli crteze:**
1. Idite u **Settings** -> **Community plugins**.
2. Isključite *Safe Mode* i kliknite na **Browse**.
3. Pretražite i instalirajte **Excalidraw**.
4. Kliknite na **Enable**.
5. *Svi `.excalidraw` i `.md` crteži će se sada automatski renderovati.*
